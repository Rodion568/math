Извлечение квадратного корня

Квадратный корень из числа a — это число b, которое при возведении в квадрат даёт a.

Обозначение:
√ 25 = 5, потому что 52 = 25.

Примеры:
√ 16 = 4
√ 100 = 10

Извлечение корня из полных квадратов

Алгоритм:

  1. Вспомнить таблицу квадратов чисел от 1 до 20.
  2. Найти число, которое при возведении в квадрат даёт подкоренное выражение.
Таблица для запоминания:

Например:

  1. √ 49 = 7
  2. √ 81 = 9
  3. √ 121 = 11

Приближённое извлечение корня

Если число — не полный квадрат:

  1. Найти ближайшие квадраты.
  2. Оценить значение между ними.
Пример: √ 50
  • Ближайшие квадраты: 72 = 49 и 82 = 64.
  • 49 < 50 < 64 → 7 < √ 50 < 8
  • Приближение: √ 50 ≈ 7.1.

Метод подбора:

  1. Попробуем 7.1: 7.12 = 50.41 – много.
  2. Попробуем 7.07: 7.072 ≈ 50 – почти точно.

Письменный метод

Алгоритм «Столбиком»:

  1. Разбить число на пары справа налево.
  2. Подобрать цифру результата пошагово.
Пример: √ 529.
  1. Разбиваем: 5 29.
  2. Ищем число, квадрат которого ≤ 5: 22 = 4.
  3. Вычитаем: 5 - 4 = 1, сносим 29 → 129.
  4. Удваиваем текущий результат (2 → 4), подбираем цифру x: (40 + x) * x ≤ 129.
    x = 3 (43 * 3 = 129).
  5. Ответ: √ 529 = 23.
